# Jpa Unit Test Example Backend Springboot

## Introduction

This is a jpa unit test made with Java 11.

### Back-end

The back-end is using an [H2 database](https://www.h2database.com) behind to store the post.

## How to build and execute it

### Requirements

+ git
+ Java 11
+ Maven 3

### Dependency

+ springboot-data-jpa
+ h2

### Run Test

```
mvn test
```
